# Achei Aventura

**Aventura sem achismo.**

Ecossistema digital de turismo e aventura da Destino Ativo Eventos e Turismo LTDA, desenvolvido para conectar viajantes, rotas verificadas, cidades e negócios locais.

## Piloto

O piloto inicial é centrado em Nova Petrópolis e inclui:

- rotas reais com arquivos GPX;
- Rota Malakoff e Rota 9 Colônias;
- fichas de segurança, dificuldade, piso, apoio e cobertura de celular;
- experiência de navegação e preparação para mapas offline;
- recomendação de rotas baseada no perfil do usuário;
- passaporte, pontuação, selos e recursos sociais;
- experiências, parceiros, trips premium e oportunidades B2B.

## Desenvolvimento

Projeto web em React/TypeScript. Para validar localmente:

```bash
npm install
npm run dev
```

Para validar a versão de produção:

```bash
npm test
```

## Situação atual

Este repositório contém o piloto comercial e funcional. A próxima fase prevê backend compartilhado entre site e aplicativo, autenticação, banco de dados, painel administrativo e navegação móvel offline.
